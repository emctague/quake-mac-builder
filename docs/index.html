App downloads are automatically generated by build scripts available at emctague/quake-mac-builder.

Build scripts patch some games to ensure a smoother user experience on macOS.

While these games are open-source, their assets are not. Game pack files (.pak, .pk3) from the official game CDs or digital downloads will need to be placed in the appropriate Application Support directories as indicated by the app before games will run.
